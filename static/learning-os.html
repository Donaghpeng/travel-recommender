每日执行机制

下午+晚上 · 2-3h
第一步
3-5 分钟
🌱 最小启动：打开昨天停下的地方
不要计划，不要热身。直接打开文件/项目/书——第一件事就是 做。
第二步
40-60 分钟
🛠️ 深度工作块 #1
做最核心的任务。
规则：如果遇到难题卡住 超过 10 分钟 → 标记障碍点 → 切换（不是放弃，是缓存进度）
间隙
5-10 分钟
🧘 主动休息（不是刷手机！）
站起来 · 走几步 · 喝水 · 看窗外
❌ 不要碰短视频 —— 一旦开刷基本回不来
第三步
40-60 分钟
🛠️ 深度工作块 #2
继续工作块#1 或切换到另一个任务（按目标优先级）
最后
5 分钟
📝 每日三行记录
写 3 条（不多不少）：
① 今天完成了什么？
② 卡在哪？
③ 明天第一步做什么？

周复盘模板

每周日 15 分钟

不是「今天学了什么」，而是「这周留下了什么」。

📊
进度快照
翻看 7 天的「每日三行记录」，数出：
✅ 完成了几件事？
❌ 卡在几个点上？
⏭️ 哪些没开始？
🔗
连接点发现
「这周学的东西，和 上周的什么有关？」
哪怕只有一个连接，也要写下来。
解决「学了很多但连不起来」
🛡️
障碍清单
这周被什么事 打断/拖住 了？
- 是内容太难？
- 是工作太累？
- 是没想清楚为什么学？
识别出来，下周就能绕过去。
🎯
下周主攻
只定 1 件事 作为下周的核心目标。
不要列3件——列了等于没列。
「这周做完这一件事就算胜利」

💡 实际操作：写在纸质笔记本的首页，每周日翻到同一页，写满 2 页纸就是一次完整的复盘。

🧠

知识归纳框架

解「看了就忘」

看完一个知识点 → 用统一格式写成卡片，每周翻一次。

📇 卡片模板（纸质/电子皆可）
┌─────────────────────────┐ │ 📌 [概念名称] │ │ 📆 2026-06-18 │ │─────────────────────────│ │ Q: 它能解决什么问题？ │ │ A: 一句话解释 │ │─────────────────────────│ │ 💡 自己的理解: │ │ 用 NJ 的话写，不是抄书 │ │─────────────────────────│ │ 🔗 连接: 它和之前学的 │ │ XX 是什么关系？ │ └─────────────────────────┘
✍️ 例子（假设在学 Cursor）
┌─────────────────────────┐ │ 📌 Cursor Rules │ │ 📆 2026-06-18 │ │─────────────────────────│ │ Q: 它做什么？ │ │ A: 告诉 AI 项目上下文， │ │ 让生成更精准 │ │─────────────────────────│ │ 💡 自己的理解: │ │ 像给新同事写 onboarding │ │ 文档——把规矩说清楚 │ │─────────────────────────│ │ 🔗 连接: 和 .cursorrules │ │ 文件是一回事 │ └─────────────────────────┘
🐾 小比提示： 你不用记住所有卡片。
关键是 「写卡片」这个动作本身就在强化记忆 —— 它逼你用自已的话把东西说一遍。
每周日翻一翻，一眼就能看到「这周学了什么」——那面镜子的作用就出来了。
📎 知识连接的技巧（针对「难以联系」）
  • 🔗 「这个像什么」 — 新知识像以前学过的什么？用类比写下来
  • 🔗 「它们是什么关系」 — A 是 B 的前置？A 和 B 是并列/互补？
  • 🔗 「这个推翻了我之前哪个认知」 — 冲突点最值得记录

防放纵机制

专治「困难+累→刷视频」
⏰
5 分钟启动法
不想学的时候，只要求自己做 5 分钟。
5 分钟后如果还想停，可以停。
实际上 99% 的情况下，5 分钟后你就进入了。
🔄
换脑策略
遇到难题卡住时，不要刷短视频。
换成：
- 整理之前的笔记
- 换一个更简单的子任务
- 或者直接站起来走 2 分钟
短视频是单行道，一旦进去就很难回头。
🧹
环境隔离
把手机放在 另一个房间。
学完才能拿回来。
意志力是靠不住的——别考验它。
❓
「为什么学」检查
如果你突然觉得 「没意义了」——
停下，问自己三个问题：
① 我当初为什么要学这个？
② 我已经会了什么？（翻卡片）
③ 再往前走一步能看到什么？
很多时候「没意义」只是因为卡住了，不是真的没意义。
📝

每日三行模板

5 分钟搞定

每天学习结束写 3 行，不多不少。这是给 NJ 的「学习镜子」。

① 今天做了什么： _________________
② 卡在了哪里： _________________
③ 明天第一步： _________________

💡 为什么只写 3 行？
因为你之前「不做笔记、做完才复盘」——3 行是门槛最低的起点。
如果一开始就设计复杂的笔记系统，你大概率不会坚持。
先养成 每天 3 行 的习惯，一个月后如果觉得不够再升级。
